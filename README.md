# Cadastro de Clientes

---

## 🎯 Objetivo

Projeto **CRUD** focado na gestão e cadastro de **clientes**, **fornecedores** e **usuários**.  
Permite **criar, ler, editar e excluir** informações, com o objetivo de melhorar a administração de uma empresa.  
Este projeto foi desenvolvido em uma situação de aprendizagem.

---

## 🛠️ Ferramentas Utilizadas

- **HTML** – Estrutura dos formulários e páginas.
- **CSS** – Estilização e layout visual.
- **SQLite** – Banco de dados leve e embutido.
- **Python + Flask** – Backend e definição das rotas e funções.

---

## ✅ Conclusão

Foi um projeto bastante desafiador na época,  
testando nossas habilidades em diversas áreas:

- **Banco de dados** com SQLite.
- **Funções e rotas** com Python e o framework Flask.
- **Formulários** com HTML.
- **Estilo visual** com CSS.

Essa experiência foi essencial para reforçar nossos conhecimentos em desenvolvimento full stack básico.
